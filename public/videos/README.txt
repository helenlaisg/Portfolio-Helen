Coloque seus arquivos de vídeo nesta pasta.

Para usar um vídeo no seu site:
1. Copie o arquivo de vídeo para esta pasta (ex: projeto1.mp4).
2. No arquivo src/components/Projects.jsx, altere a linha 'videoUrl' do projeto correspondente:

De:
videoUrl: 'PLACEHOLDER_VIDEO_URL'

Para:
videoUrl: '/videos/projeto1.mp4'

Formatos recomendados: .mp4, .webm
